 CLAUDE CODE - INSTALACE NA NOVE WINDOWS 10 / 11 PC
 (pro uplne zacatecniky)
============================================================

CO TENTO BALIK UDELA
--------------------
  - Nainstaluje Git for Windows
  - Nainstaluje Claude Code (Node.js neni potreba)
  - Vytvori na PLOSE velkou ikonu "CLAUDE"
    -> uzivatelka pak jen 2x klikne a Claude bezi (zadny terminal)


JAK NA TO (udelas TY, jako sprava)
----------------------------------
1) Zkopiruj celou tuto slozku na nove PC (nebo spust z flesky).

2) Pravym tlacitkem na  INSTALUJ.bat  -> "Spustit jako spravce".

3) Pockej, az instalator dobehne.

4) DULEZITE - prihlas je predem:
   2x klikni na ikonu CLAUDE na plose a projdi prihlasenim
   k Anthropic uctu. Tim padem uzivatelky uz jen pisou.


JAK TO POUZIVAJI UZIVATELKY (zacatecnice)
-----------------------------------------
   1) 2x kliknout na ikonu CLAUDE na plose
   2) Napsat normalni vetu (cesky)
   3) Zmacknout ENTER
   4) Cist odpoved, pripadne psat dal

   -> Viz vytisteny TAHAK_pro_tisk.txt (dej ho kazde k PC).


KDYZ NECO NEJDE
---------------
- "claude" nebezi po dvojkliku? -> Restartuj PC a zkus znovu.
- Stary Windows bez "winget"?   -> Instalator si stahne vse sam,
                                    staci pripojeni k internetu.
- SmartScreen blokuje .bat?      -> "Vice informaci" -> "Presto spustit".
